## Ciao – Saluto informale italiano  

- **Origine**: deriva dal veneziano *s‑cia(v)o* o *s‑ciavo*, che significava “schiavo”. Era un modo di esprimere rispetto e lealtà verso la persona a cui ci si rivolgeva【2】【3】.  
- **Evoluzione**: da forma servile è passato a diventare il più comune saluto informale in tutta Italia, usato sia per incontrarsi che per congedarsi【1】.  
- **Diffusione internazionale**: “ciao” è ormai riconosciuto in molte lingue al di fuori dell’Italia, diventando un saluto quasi universale【5】.

### Curiosità aggiuntive  
- In statistica, una **prova** è una verifica di requisiti su un campione di dati.  
- In medicina, la **prova** indica un esame volto a valutare la funzionalità di un organo o di un apparato.  
- Il verbo *provare* può anche significare “sentire” (es. *provare dolore*).

## Test – Definizione, tipologie e utilizzo  

### Definizione generale  
**Test** (sostantivo maschile) – Prova, sperimentazione empirica finalizzata a verificare una caratteristica, una capacità o un risultato【4†L1-L3】.  

### Tipologie di test più comuni  

| Tipo di test | Ambito di utilizzo | Esempio pratico | Note |
|--------------|-------------------|----------------|------|
| **Test di velocità Internet** | Reti di telecomunicazioni, ISP, utenti domestici | *Speedtest di Ookla* misura download, upload e ping in pochi secondi【1†L1-L3】 | Usato per verificare la qualità della connessione a casa o in ufficio. |
| **Test di velocità ISP (EOLO)** | Provider di banda larga, clienti EOLO | *EOLO Speed Test* consente di misurare le prestazioni della propria linea EOLO【2†L1-L3】 | Stesso principio di base di altri test di velocità. |
| **Test di velocità “Fast.com”** | Utenti che cercano una verifica rapida e senza registrazione | *Fast.com* fornisce una stima della velocità di download del proprio ISP in pochi secondi【3†L1-L3】 | Ideale per risultati immediati senza grafici complessi. |
| **Test di misurazione certificata (Agcom)** | Autorità di regolamentazione, consumatori | *Misurainternet* è la piattaforma Agcom con software certificato per misurare le prestazioni delle connessioni a banda larga da postazioni fisse【5†L1-L3】 | Utilizzato per verifiche ufficiali e reclami verso l’ISP. |
| **Test psicometrici / di valutazione** | Risorse umane, orientamento scolastico, ricerca | Test di intelligenza, test di personalità, test di abilità specifiche | Somministrati in forma cartacea o digitale, con punteggi standardizzati. |
| **Test medici** | Diagnostica, screening | Esami del sangue, test di gravidanza, test di funzionalità polmonare | Scopi di rilevare patologie o monitorare condizioni di salute. |
| **Test di prodotto** | Industria, sviluppo software, hardware | Test di usabilità, test di stress, test di compatibilità | Garantiscono qualità e sicurezza prima della commercializzazione. |

### Come funziona un test di velocità Internet (esempio pratico)  

1. **Connessione al server di test** – Il dispositivo stabilisce una connessione a un server geograficamente vicino per ridurre la latenza.  
2. **Misurazione del download** – Il server invia un grande volume di dati al dispositivo; il tempo impiegato determina la velocità di download (Mbps).  
3. **Misurazione dell’upload** – Il dispositivo invia dati al server; il tempo impiegato determina la velocità di upload.  
4. **Ping/latency** – Viene calcolato il tempo di andata‑ritorno (RTT) di piccoli pacchetti, espresso in millisecondi (ms).  

I risultati tipici includono tre valori: **Download**, **Upload** e **Ping**. Se i valori sono significativamente più bassi rispetto a quelli contrattuali, è possibile contattare l’ISP con le prove raccolte.  

### Quando è utile fare un test  

| Situazione | Perché è consigliato |
|------------|----------------------|
| **Rete domestica lenta** | Verificare se il problema è legato al proprio ISP o alla rete locale. |
| **Prima di cambiare provider** | Confrontare le velocità offerte da diversi ISP nella propria zona. |
| **Reclamo all’ISP** | Fornire dati oggettivi (download/upload/ping) per supportare la richiesta. |
| **Ottimizzazione della rete** | Identificare colli di bottiglia e valutare l’impatto di eventuali upgrade hardware o configurazioni. |
